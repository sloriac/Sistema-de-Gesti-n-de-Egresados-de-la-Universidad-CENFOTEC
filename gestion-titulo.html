Gestión de títulos

Complete todos los campos obligatorios para registrar un título académico.

*
Ingrese el código único del título, por ejemplo: BTIC-025
*
Ingrese el nombre completo del título tal como se otorga, por ejemplo: Bachillerato en tecnologías de la información y comunicación
*
Ingrese el código de la carrera a la que pertenece este título, por ejemplo: CAR-015
*
Ingrese el nivel o grado académico que otorga este título, por ejemplo: Diplomado, Bachillerato, Licenciatura, Maestría o Doctorado.
*
Ingrese la cantidad de créditos requeridos para optar por este título, por ejemplo: 160
*
Ingrese la fecha desde la cual el título se encuentra vigente, en formato día/mes/año.
*
Indique si el título se encuentra vigente o descontinuado.
